WhiteSpace 是 Flex 布局的控件之一。

## API ( 适用平台：WEB、React-Native )

| 成员        | 说明           | 类型            | 默认值       |
|------------|----------------|----------------|--------------|
| size       |  上下留白的间距，可选`xs`,`sm`,`md`,`lg`,`xl`  | string | `md`  |
