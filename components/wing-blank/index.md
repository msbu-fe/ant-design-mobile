WingBlank 是 Flex 布局的控件之一。

## API ( 适用平台：WEB、React-Native )

| 成员        | 说明           | 类型          | 默认值       |
|------------|----------------|--------------|--------------|
| size    | 两翼留白的间距，可选`sm`,`md`,`lg`  | string |  `lg`  |
